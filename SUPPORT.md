## Reporting Issues

Before opening an issue:

1. Check existing [issues](https://github.com/knirski/paperless-ingestion-bot/issues) for similar reports
2. Ensure you're on the latest release
3. Include: Node.js version, config (redact secrets), steps to reproduce, and error output

Use the [bug report](https://github.com/knirski/paperless-ingestion-bot/issues/new?template=bug_report.md) or [feature request](https://github.com/knirski/paperless-ingestion-bot/issues/new?template=feature_request.md) templates when creating issues.

## Security

For security vulnerabilities, see [SECURITY.md](SECURITY.md). Do not report security issues in public issues.
