# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the paperless-ingestion-bot project.

## Format

Each ADR is a markdown file named `NNNN-title-with-dashes.md` where NNNN is a zero-padded number. See [adr-template.md](adr-template.md) for the template.
